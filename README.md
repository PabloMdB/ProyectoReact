
# Proyecto final - CoderHouse

Este proyecto es una SPA de ReactJS hecha con VITE y deployada en Nestlify.


## Instalacion

Para instalar el proyecto, lo descargamos y luego lo ejecutamos.

```bash
  npm install 
  npm run dev
```
